# Welcome to NCI Biobank documentation

This site was built using the [React](https://reactjs.org/) front-end library and [Create React App](https://github.com/facebook/create-react-app) as the foundation. The [Material-UI](https://material-ui.com/) component library is also a core package for this site. Our custom components are integrated with Material-UI components and we make use of Material-UI's styling solution. We've implemented [react-i18next](https://react.i18next.com/) to support language translations, and currently support both English and Spanish languages. All of these features come together to create a fully responsive, accessible and performant single page application (SPA).

This documentation site is powered by [Storybook](https://storybook.js.org). Storybook helps us build UI components in isolation from our app's business logic, data, and context.
That makes it easy to develop hard-to-reach states. Save these UI states as **stories** to revisit during development, testing, or QA. Browse stories now by navigating to them in the sidebar.
